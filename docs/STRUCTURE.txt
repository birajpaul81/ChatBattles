Getting Started
===============
1. npm install
2. cp .env.local.example .env.local
3. Edit .env.local with your API keys
4. npm run dev
5. Visit http://localhost:3000


Deploy to Production
====================
1. Push to GitHub
2. Import to Vercel
3. Add environment variables
4. Deploy!
5. Update Clerk redirect URLs
